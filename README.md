# ConsoleBlackJack

## Описание проекта

**ConsoleBlackJack** — это консольная реализация популярной карточной игры "BlackJack". Игра предоставляет возможность играть в одиночном режиме или против компьютера.

## Функциональность

- **Игровые режимы**:
  - **Одиночный режим**: цель - набрать 21 очко.
  - **PvC (игрок против компьютера)**: цель - набрать больше очков, чем компьютер, но не превышая 21.

- **Карточная механика**:
  - Генерация и перемешивание колоды.
  - Раздача карт игроку и дилеру.
  - Подсчет очков и определение победителя.

## Основные классы

### `Player`

Класс представляет игрока и его действия.

#### Методы:

- **`Player(std::string PlayerName)`**  
  Конструктор, инициализирующий игрока с заданным именем.

- **`std::string GetPlayer()`**  
  Возвращает имя игрока.

- **`void TakeCard(std::string suit, std::string value)`**  
  Добавляет карту в руку игрока.

- **`void ShowHand()`**  
  Отображает карты в руке игрока.

- **`int CountHandValue()`**  
  Считает сумму очков в руке игрока.

---

### `Deck`

Класс для управления колодой карт.

#### Методы:

- **`void SetDeck()`**  
  Инициализирует колоду.

- **`void Shuffle()`**  
  Перемешивает колоду и выбирает случайную карту.

- **`std::array<std::string, 2> GetCard()`**  
  Возвращает текущую выбранную карту и удаляет её из колоды.

- **`int CountDeckSize()`**  
  Подсчитывает количество оставшихся карт в колоде.

---

### `BlackJack`

Класс, управляющий игровым процессом.

#### Методы:

- **`void Run()`**  
  Запускает игру, отображает главное меню.

- **`void InitMenu()`**  
  Отображает меню для выбора режима игры.

- **`void ChooseGameMode()`**  
  Позволяет выбрать между одиночным режимом и PvC.

- **`void StartGameAlone()`**  
  Запускает одиночную игру.

- **`void StartGamePvC()`**  
  Запускает игру против компьютера.

- **`std::string GreetingInGame()`**  
  Запрашивает имя игрока и отображает правила.

- **`void ShowGameStartText()`**  
  Отображает обратный отсчет до начала игры.

- **`void EndGame()`**  
  Завершает текущую игру с возможностью перезапуска.

- **`int TryInput()`**  
  Обрабатывает ввод пользователя.

---

## Файлы проекта

- **`Player.h` и `Player.cpp`**  
  Реализация класса игрока.

- **`Deck.h` и `Deck.cpp`**  
  Реализация класса колоды карт.

- **`BlackJack.h` и `BlackJack.cpp`**  
  Реализация игрового процесса.

- **`ConsoleBlackJack.cpp`**  
  Точка входа в программу.

---

## Компиляция и запуск

1. Убедитесь, что у вас установлен компилятор C++.
2. Скомпилируйте проект с помощью команды:
   ```bash
   g++ ConsoleBlackJack.cpp BlackJack.cpp Player.cpp Deck.cpp -o ConsoleBlackJack
   ```
3. Запустите исполняемый файл:
	```bash
   ./ConsoleBlackJack
   ```
   
---

## Примечания
- Программа работает на Windows. Для Linux/macOS следует заменить вызов `system("cls")` на `system("clear")`.
- В игре используется функция `Sleep()`, которая требует библиотеки `Windows.h`. На других ОС ее можно заменить аналогами из стандартной библиотеки.

# ConsoleMemoryGame

## Описание проекта

**ConsoleMemoryGame** — это консольная игра на тренировку памяти, в которой игроку предлагается запоминать и воспроизводить числа или их последовательности. Игра включает три режима, каждый из которых ориентирован на развитие памяти и концентрации.

## Функциональность

- **Игровые режимы**:
  - **Число**: игроку нужно запомнить и воспроизвести одно случайное число.
  - **Последовательность**: запоминание и ввод последовательности чисел.
  - **Матрица**: воспроизведение матрицы чисел после её исчезновения.

- **Параметры игры**:
  - Установка диапазона случайных чисел.
  - Выбор количества раундов.
  - Отслеживание количества правильных и ошибочных ответов.

## Основные классы

### `MemoryGame`

Класс отвечает за управление игровым процессом, предоставляя методы для инициализации игры, выбора режима и обработки результатов.

#### Методы:

- **`void RunGame()`**  
  Запускает игру и отображает приветственное сообщение.

- **`void InitMenu()`**  
  Инициализирует параметры игры: диапазон чисел, количество раундов и выбор игрового режима.

- **`void CheckInput(std::string gameMode)`**  
  Проверяет корректность выбранного режима игры и запускает соответствующий режим.

- **`void EndGame()`**  
  Выводит результаты игры и предлагает начать новую сессию или завершить программу.

---

### Режимы игры

#### **`void runNumbMode()`**  
Игровой режим "Число".  
- Генерируется одно случайное число.
- Игрок вводит запомненное число.  

#### **`void runSequenceMode()`**  
Игровой режим "Последовательность".  
- Генерируется последовательность чисел заданной длины.  
- Игрок вводит последовательность в правильном порядке.  

#### **`void runMatrixMode()`**  
Игровой режим "Матрица".  
- Генерируется матрица чисел заданных размеров.  
- Игрок воспроизводит матрицу после её исчезновения.  

---

### Вспомогательные методы

- **`int createNumb()`**  
  Генерирует случайное число в заданном диапазоне `[x0, y0]`.

- **`std::vector<int> createNumbSequence()`**  
  Возвращает последовательность случайных чисел длиной `nSize`.

- **`std::vector<std::vector<int>> createMatrix()`**  
  Генерирует матрицу чисел размером `nSize x mSize`.

- **`void formatText(std::string mode)`**  
  Отображает обратный отсчет и очищает экран перед вводом ответа.

- **`void formatMode(std::string mode)`**  
  Отображает обратный отсчет перед началом игры в выбранном режиме.

---

## Файлы проекта

- **`MemoryGame.h` и `MemoryGame.cpp`**  
  Реализация класса `MemoryGame` и всех его методов.

- **`ConsoleMemoryGame.cpp`**  
  Точка входа в программу.

---

## Компиляция и запуск

1. Убедитесь, что установлен компилятор C++.
2. Скомпилируйте проект с помощью команды:
   ```bash
   g++ ConsoleMemoryGame.cpp MemoryGame.cpp -o ConsoleMemoryGame
   ```
3. Запустите скомпилированный файл:
   ```bash
   ./ConsoleMemoryGame
   ```

# ConsoleSimpleCalc

## Описание проекта

**ConsoleSimpleCalc** — это простой консольный калькулятор, который выполняет базовые арифметические операции. Программа позволяет пользователю вводить математическое выражение в формате `a + b`, где `a` и `b` — числа, а оператор — один из следующих: `+`, `-`, `*`, `/`, `%`.

## Функциональность

- **Базовые операции**:
  - **Сложение** (`+`)
  - **Вычитание** (`-`)
  - **Умножение** (`*`)
  - **Деление** (`/`)
  - **Вычисление процента** (`%`)

- **Особенности**:
  - Проверка деления на ноль.
  - Возможность многократного выполнения операций в рамках одной сессии.
  - Программа завершает работу при вводе соответствующей команды.

## Основные классы

### `Calculations`

Класс предоставляет метод для выполнения арифметических операций.

#### Методы:

- **`double Calculate(double x, char operation, double y)`**  
  Выполняет математическую операцию над двумя числами `x` и `y` в зависимости от переданного оператора `operation`.

  **Параметры**:
  - `x` — первое число.
  - `operation` — оператор (`+`, `-`, `*`, `/`, `%`).
  - `y` — второе число.

  **Возвращает**:
  - Результат выполнения операции. В случае неверного оператора возвращается `0.0`.

---

## Файлы проекта

- **`Calculations.h` и `Calculations.cpp`**  
  Реализация класса `Calculations`.

- **`ConsoleSimpleCalc.cpp`**  
  Точка входа в программу. Реализует взаимодействие с пользователем и вызов методов класса `Calculations`.

---

## Компиляция и запуск

1. Убедитесь, что установлен компилятор C++.
2. Скомпилируйте проект с помощью команды:
   ```bash
   g++ ConsoleSimpleCalc.cpp Calculations.cpp -o ConsoleSimpleCalc
   ```
3. Запустите скомпилированный файл:
   ```bash
   ./ConsoleSimpleCalc
   ```

# ConsoleSnakeGame

## Описание проекта

**ConsoleSnakeGame** — это консольная версия классической игры "Змейка". Игрок управляет змейкой, собирая яблоки на игровом поле и избегая столкновений с границами или своим хвостом.

## Функциональность

- **Управление**:
  - Поддержка управления клавишами `W`, `A`, `S`, `D` или стрелками.
- **Механика игры**:
  - Динамическое перемещение змейки.
  - Сбор яблок для увеличения длины змейки.
  - Проверка на столкновение с границами и своим телом.
- **Игровое поле**:
  - Генерация игрового поля с границами.
  - Случайное появление яблок.

## Основные классы

### `Field`

Класс отвечает за создание и отображение игрового поля.


---

### `Snake`

Класс управляет движением змейки и ее взаимодействием с игровым полем.

---

### `SnakeGame`

Класс реализует основную логику игры

---
